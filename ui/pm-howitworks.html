How it works!

Technology

For database we use Apache CouchDB, JSON for documents and JavaScript for MapReduce queries.

Create document

Jsonbridje and WDK.API.CouchDb 

            var user = new model.user();

            user.username = $('#username').val();
            user.password = $('#password').val();
            user.email = $('#useremail').val();
            user.position = $('#status').val();
            
            JsonBridge.execute('WDK.API.CouchDb', 'createDocument', 
            ['pmsystem', JSON.stringify(user)], function (data) {
				
				$.cookie('user.Id', data.result.Id);

            });
